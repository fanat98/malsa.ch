# malsa.ch

Personal CV and notes website for Aslam Idrisov.

## Stack

- Astro
- TypeScript
- Markdown-ready content structure

## Commands

```bash
npm install
npm run dev
npm run build
```

## Deployment

The site builds to `dist/` and can be deployed to GitHub Pages, Netlify, Vercel, Cloudflare Pages, or a static host.
